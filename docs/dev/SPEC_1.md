# Disco Elysium

Чат бот в бизнес-аккаунте, который отвечает характеристиками и диалогами.
Это своеобразная игра, где собеседник может поговорить с характеристиками

## Идея проекта

- Чат бот, который общается с тобой от лица эмоций
- загрузить текст/картинки/кружочки/гс
- набор ответов от разных эмоций как в диско элизиум
- Для каждой эмоции бросается 2 кубика (сложность эмоции определяет, насколько много должно выпасть) - показывать успех или нет 
- Ради интереса генерировать из исходной ситуации диалог
- в рамках диалога могут быть проверки. Провал проверки ведёт никуда 
- Нужно давать варианты ответа кнопками под сообщением и свой ответ
- Важно, чтобы кнопки можно было создавать также и самому парся сообщение
- Эта игра должна работать в рамках существующего чата с конкретным пользователем (в базе это нужно задать)

Создавать варианты диалога или отвечать от лица характеристик не всегда нужно - они должны быть подходящими под диалог

Мы не будем использовать картинки, мы будем использовать эмодзи и названием эмоции капслоком и курсивом

Посмотри доки из /docs/references

В качестве fallback диалог должен создаваться по описанию по команде в диалоге

## Технологии

- python как язык программирования
- aiogram библиотека для создания телеграм бота
- FastAPI сервер
- PostgreSQL - база данных диалогов, скиллов и юзеров
- Redis для состояния диалога
- Tortoise - ORM и миграции
- разные API для всего остального по необходимости:
  - chatGPT для текста
  - что-то для транскрипта ГС и кружочков
  - анализ картинок (тоже openAI, мб)

Все запросы структурированные
