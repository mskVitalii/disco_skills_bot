# Характеристики Disco Skills Bot

Каждая характеристика — это голос внутри, комментирующий происходящее.

**Формат в чате:**
```
{emoji} НАЗВАНИЕ
Текст ответа (1–3 предложения)

🎲 [d1+d2] + уровень = итого vs сложность — РЕЗУЛЬТАТ
```

## Механика проверок

### Типы проверок

**Пассивная проверка** — характеристика сама вставляет комментарий без броска.

**Активная проверка** — бросается 2d6 + уровень навыка vs сложность:

| Результат | Условие |
|-----------|---------|
| ⚀⚀ КРИТИЧЕСКИЙ ПРОВАЛ | Дубль единиц — всегда провал |
| ⚅⚅ КРИТИЧЕСКИЙ УСПЕХ | Дубль шестёрок — всегда успех |
| ✅ УСПЕХ | Итого ≥ Сложность |
| ❌ ПРОВАЛ | Итого < Сложность |

### Уровни сложности

| Уровень | Очки | Нужный уровень |
|---------|------|----------------|
| Тривиальная | 6–7 | 0–1 |
| Лёгкая | 8–9 | 2–3 |
| Средняя | 10–11 | 4–5 |
| Сложная | 12 | 6 |
| Грозная | 13 | 7 |
| Легендарная | 14 | 8 |
| Героическая | 15 | 9 |
| Божественная | 16–17 | 10–11 |
| Невозможная | 18–20 | 12–14 |
